普段はネットワークサービス開発をしています。 ツール作成のためにPythonを使ったり、ルータのコンフィグを作成したりしています。 使っている主なネットワーク技術はMPLS, NETCONF, SNMPです。トラフィックエンジニアリングを活用したVPN構築などを行っています。

Haskellというプログラミング言語が好きなので、このサイトの記事のほとんどはHaskellか関数型プログラミングに関する内容になる予定です。

I usually write code or router configuration to build network services for production infrastructure. The main technologies that I've ever used are MPLS, NETCONF, SNMP. They are essential building blocks to provide reactive, robust network.

This site contains posts about Functional Programming and Software Defined Everything. Most posts are written in Japanese.

I really like Haskell that is the most excellent and elegant programming language I have ever seen. So code examples in post about Functional Programming are written in Haskell.
