Posted on January 19, 2015

I’m developing haskell app with nix package manager on OSX.

I added $HOME/.nixpkgs/config.nix to override package definitions. These package are deployed on Hackage only (not Nixpkgs repo) such as HDBC-session.

Then I ran nix-shell on my local project depends on those packages, but evaluation of nix expression was failed because nix could not find package names.

How can I import overriden packages by user when I run nix-shell?
